Vještine

Ovdje su izloženi dosadašnji programski jezici i biblioteke s kojima sam upoznat te sposoban za primjenu:

HTML5

HyperText Markup Language. Prezentacijski jezik za izradu web stranica.

CSS3

Cascading Style Sheets. Stilski jezik, koji se rabi za opis prezentacije dokumenta napisanog pomoću markup-a.

JavaScript

Skriptni programski jezik, koji se izvršava u web pregledniku na strani korisnika.

Bootstrap

Front-end framework. Omogućava jednostavan razvoj mobilnih web aplikacija.

SASS

Pretprocesorski skriptni jezik koji se kompajlira u CSS. Omogućava lako uređivanje stila programa.

React

Front-end biblioteka za JavaScript. Koristi se za izgradnju korisničkog sučelja. Naglasak joj je na optimizaciji.

Redux

Biblioteka za JavaScript. Služi za upravljanje stanjem aplikacija.

JQuery

Biblioteka za JavaScript. Pojednostavljuje korištenje JS u aplikaciji. Višelinijske metode u JS JQuery prepakira u metode koje možemo pozvati u jednoj liniji koda.

Git

Softver za kontrolu verzije programa.

Java

Java je objektno orijentirani programski jezik.
